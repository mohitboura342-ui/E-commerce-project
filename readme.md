# Advanced Multivariate Customer Churn Analysis

This project performs an exploratory data analysis (EDA) on customer behavior to understand the major factors contributing to churn.  
The analysis focuses on discount usage, price sensitivity, customer satisfaction, cart abandonment, product reviews, return rate, and customer support activity.

## Project Objective

The main goal of this project is to identify patterns that influence customer churn and find business insights that can help improve customer retention.

## Key Features of the Analysis

- Customer pricing behavior analysis
- Discount usage vs price sensitivity study
- Cart abandonment pattern analysis
- Return rate comparison by churn status
- Product review score comparison
- Customer support tickets and satisfaction correlation
- Multivariate correlation heatmap

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Visualizations Included

- Heatmap for customer pricing behavior
- Scatter plot for discount dependency vs price sensitivity
- Boxplot for product review score by churn
- Violin plot for return rate distribution by churn
- Correlation heatmap for support tickets, satisfaction, engagement, and churn-related variables

## Key Insights

- Customers with higher discount usage show strong price sensitivity.
- Churned customers generally have lower satisfaction and slightly higher return rates.
- Product review scores are lower for churned customers compared to retained customers.
- More customer support tickets are strongly linked with lower satisfaction.
- Cart abandonment and churn-related behavior indicate weak but important engagement issues.

## Business Conclusion

The analysis shows that customer churn is strongly connected to low satisfaction, higher return activity, and frequent support issues.  
Customers who rely heavily on discounts are more price-sensitive, which increases churn risk when offers are reduced.  
Businesses should improve customer support, product quality, and retention strategies to reduce churn and increase loyalty.
